    General Page Structure

    The page is divided into two main sections! The content part and the sidebar part! 

    Page Content

    The following structure applies for the content! 

    <div class="page-content">
    <div class="content">
        <div class="container">
            <!--Whatever you like here, with a 20 pixel margin bottom-->
        <div>
        <div class="decoration"></div>
        <div class="container no-bottom">
            <!--Whatever you want here with no margin bottom!-->
        </div>
    </div>
</div>

    The no-bottom half-bottom are classes that apply to EVERYTHING in the template! No bottom removes the bottom margin from the container, half-bottom only adds 10 pixels margin bottom and not adding it leaves the default 20 pixels margin bottom!

	Copying this code into your page-content class will generate a sidebar. How it works is pretty easy! Don't be scared by teh size! 
    

    
	First achor classes are responsible for the phone, sms and closing of the sidebar! 
    

    
	Beeneth the first anchor classes is the sidebar logo! To change the logo width when adding your own logo, search for the sidebar-logo and logo classes inside of style.css
    

    
	The menu item class, creates a simgle menu when used in the bellow syntax. 
    

    

    
	
              <div class="menu-item">
    

    

    
	              <strong class="home-icon"></strong>
    

    
	              <a class="menu-enabled" href="index.html">Home</a>
    

    
	          </div> 
    

    
	If you wish to create a submenu, use the bellow syntax
    

    

    
	
              <div class="menu-item">
    

    

    
	              <strong class="features-icon"></strong>
    

    
	              <a class="menu-disabled deploy-submenu" href="#">Features</a>
    

    
	              <div class="clear"></div>
    

    
	              <div class="submenu submenu-on-page">
    

    
	                 <a href="intro.html">The Drawer Introduction</a>     <em class="submenu-decoration"></em>
    

    
	                 <a class="submenu-active" href="other.html">Other Features</a>
    

    
	              </div>
    

    
	          </div> 
    

    
	This syntax creates a submenu! 
    

    
	If you wish your menu to be selected, meaning have the right side ball white, replace menu-disabled with menu-enabled.
    

    
	If you wish your sidemenu to be active, the submenu class will be <div class="submenu submenu-on-page">, by this, the submenu will stay deployed! 
    

    
	If you wish to make a submenu from the sidebar active, simply give one of the anchors the class <a class="submenu-active"... >
    

    
	The above submenu has both of these examples included! 
    

    
	The sidebar heading is a text that spans full width of the screen and is bolded white!
